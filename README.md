## 4. Economic Model

### 4.1 Token Overview
- **Native Asset:** XP1
- **Max Supply:** 1,000,000,000 XP1
- **Divisibility:** 8 decimal places, enabling micro-transactions
- **Philosophy:** Scarcity, elegance, and intentionality for an elite blockchain ecosystem

### 4.2 Block Reward Distribution
Each block reward is split to reflect fairness, equality, and network sustainability:

| Role                   | Share | Notes |
|------------------------|-------|-------|
| Miners / Validators    | 60%   | Rewards proportional to network contribution |
| Smartnodes / Stakers   | 25%   | Incentivizes decentralization and network stability |
| Dev / Creator & Ecosystem | 10% | Split: 5% Dev, 3% Community, 2% Ecosystem for sustainable development |
| Community Micro-grants | 5%    | Direct support for small participants, adoption, and humanitarian applications |

**Philosophy:** This distribution ensures that contributors are fairly compensated while preserving equality, freedom, and sustainability. AI-driven dynamic issuance adjusts supply and rewards over time, maintaining long-term network stability.
